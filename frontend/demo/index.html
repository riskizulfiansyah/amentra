Amentra

Intelligence that adapts to your context

Usage

Add the <amentra-widget> element to any page with the required app-id attribute.

<amentra-widget app-id="my-app" api-base-url="http://localhost:8080"></amentra-widget>

Try it yourself

Make sure the backend is running on localhost:8080. Click the chat bubble below to start.

Attributes
